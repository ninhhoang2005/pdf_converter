TRANSLATION GUIDE - PDF CONVERTER
===================================

This application uses the industry-standard Gettext translation system.

HOW TO TRANSLATE (RECOMMENDED)
------------------------------

1.  **Install Poedit**: Download and install Poedit (https://poedit.net/), a free editor for translation files.
2.  **Open the Template**:
    -   Go to the `locales` folder.
    -   Open `messages.pot` with Poedit.
3.  **Create New Translation**:
    -   In Poedit, click "Create new translation".
    -   Select your language (e.g., French, Japanese).
4.  **Translate**:
    -   Poedit will show you the list of English strings.
    -   Click on a string and type the translation in the bottom box.
5.  **Save**:
    -   Save the file. Poedit will ask where to save.
    -   Navigate to `locales` folder.
    -   Create a folder for your language code (e.g., `fr` for French).
    -   Inside that, create a folder named `LC_MESSAGES`.
    -   Save the file as `messages.po` inside `LC_MESSAGES`.
    -   **Important**: Poedit will automatically generate a `messages.mo` file next to it. This binary file is what the app actually uses.

    Structure Example:
    locales/
    ├── messages.pot
    ├── vi/
    │   └── LC_MESSAGES/
    │       ├── messages.po
    │       └── messages.mo
    └── fr/
        └── LC_MESSAGES/
            ├── messages.po
            └── messages.mo

6.  **Activate**:
    -   Open PDF Converter.
    -   Press F4 (Options).
    -   Output language should now be selectable.
    -   Select and Restart.

UPDATING TRANSLATIONS
---------------------
If the application updates and has new strings:
1.  Open your existing `.po` file in Poedit.
2.  Go to **Translation** -> **Update from POT file...**
3.  Select the updated `locales/messages.pot`.
4.  New strings will appear for you to translate.

ADVANCED (Command Line)
-----------------------
If you don't use Poedit, you need to compile `.po` to `.mo` manually.
We have included a helper script:
    python tools_i18n.py
This script will scan the code to update `messages.pot` and verify existing files.
